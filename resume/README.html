Intro

此简历模板：

  • 内容上，来源于我本人真实求职经历以及参加多次网上简历培训总结而成
  • 风格上，参考众多简历风格，@DIYgod，@ekCit以及我自己的博客主题 itsay.me

欢迎提issue或者star，ღ( ´･ᴗ･` )比心

Object

前端求职/程序员求职

Usage

  1. 先Star/Fork本项目，然后Clone或者直接下载到本地
  2. 修改index.html内相关信息
  3. 微调样式（作为前端求职，这点应该不成问题）
  4. 生成pdf(开发中)
  5. 部署到线上
  6. 生成访问二维码（开发中）
  7. 祝您求职成功！
